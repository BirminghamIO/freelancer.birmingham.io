About

This website was created by Mariana Morris.

As a designer, I regularly need to hire freelance designers when things get too busy in the office. Although I have a list of designers I regularly use, it is not always up to date or sometime I need designers with other skills.

I know this is not only a problem I face, since I have been asked many times for recommendations from other agencies and start ups companies.

So, I decided to create this list to make it easier for companies to find design freelancers.

This website is under development

This website was built as a Minimal Viable Product. I want to test it to see if it is going to be helpful to other people and if people are going to use it before spending more time on it.

At the moment it only lists designers and other professionals that might be part of a web design team, including copywriters and front-end developers. Again, this is a test and if you think other skills are missing from the list, let me know.

Your feedback would be very helpful. Please, get in touch.
