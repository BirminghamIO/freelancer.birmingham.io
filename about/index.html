About

This project is maintained by Birmingham.IO - The tech scene community for Birmingham.

Birmingham.IO is a collection of community projects that bring the tech scene of Birmingham together.

The Birmingham Freelancers project aims to make the task of connecting the right freelancer with the right project easier by providing a filterable list of all the freelancers in the area.

If you're looking for a freelancer, browse the list. If you're a freelancer, submit your details.

Open source

This project is open source and open to forking and pull requests on GitHub. Visit the official repository for more information.
